Spark 终端快速开发

2018-04-25 11:20:00 | 技术

举例说明

  • 使用场景： 为了快速响应开发，分析 HDFS 上存储的日志内容
  • 使用手段： 使用 spark shell 终端编写简单代码，将 HDFS 的内容解析入 Hive 库中，供查询开发
  • 准备条件： 具备 spark 终端的开发机，且能够直连 Hive

具体操作

  1. 进入 Spark 终端

  2. 加载 HDFS 上的文件

  3. 将存有内容的 RDD 注册成临时表，即可像 SQL 中的表一样使用

  4. 使用 SQL 方式取出想要的信息

  5. 保存结果
