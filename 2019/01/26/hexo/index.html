百度 & 谷歌收录问题

  • 百度

    1. https://ziyuan.baidu.com/site/index 网站上填写注册信息
    2. 验证方式：建议采用 CNAME 方式 ，其他两种方式折腾了很久都不成功，说是和 github 禁百度有关
    3. 在域名注册的网站上添加解析信息()

  • 谷歌收录

    1. https://www.google.com/webmasters/tools/home?hl=zh-CN 网站上填写注册信息
    2. 验证方式:建议采用 html 验证的方式
    3. 将下载好的 html 文件移动到 hexo 根目录的 source 目录下
